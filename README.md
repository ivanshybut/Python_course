# Python course

* Синтаксис Python
1. Условные выражения (if-else):
```python
x = 5
if x > 0:
    print("Число положительное")
elif x < 0:
    print("Число отрицательное")
else:
    print("Число равно нулю")
```
2. Циклы (for и while):
```python
   # Цикл for
fruits = ["яблоко", "банан", "апельсин"]

for fruit in fruits:
    print(fruit)

# Цикл while
count = 0

while count < 5:
    print(count)
    count += 1
```
